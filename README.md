## PROJECT OVERVIEW
The Sweet Treats Bakery Website is a simple multi-page website created using HTML. It is designed to represent an online bakery where users can view different baked products and navigate between pages such as Home,Products,About and contacts. The purpose of this project is to demonstrate basic web development skills using HTML and to create a structured user friendly website layout

## WEBSITE GOALS AND MILESTONES


The project will be completed over a period of approximately 6 weeks, divided into the following 
phases: 
• Week 1: Planning and research 
• Week 2: Design (layout, colors, structure) 
• Week 3–4: Website development (coding and building pages) 
• Week 5: Testing (fixing errors, improving performance) 
• Week 6: Deployment and launch
